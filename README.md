5. Fine-tuning моделей с LoRA — эффективное дообучение LLM
Набор скриптов для тонкой настройки языковых моделей с использованием LoRA (Low-Rank Adaptation) на CPU и GPU.

🔧 Стек технологий:

PyTorch — основа для обучения

HuggingFace Transformers — загрузка моделей и токенизация

PEFT (LoRA) — эффективное дообучение без полной перенастройки

bitsandbytes — 4-битное квантование для экономии памяти

Python 3.12+ с поддержкой CPU и CUDA

✨ Ключевые особенности:

Кроссплатформенность: работает на Windows, macOS (CPU) и Linux (GPU)

Гибкая настройка: поддержка CPU, GPU, автоматический выбор устройства

Разные форматы датасетов: JSON/JSONL с полями text, instruction/output, prompt/completion

Оптимизация для слабого железа: уменьшение batch size, градиентное накопление

Чистый код: подробные комментарии и примеры запуска

Примеры моделей: DialoGPT-small, distilgpt2, gpt2, tiny-gpt2 для быстрого тестирования

🎯 Решаемые задачи:
